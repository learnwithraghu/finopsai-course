# Interactive HTML Concepts

## 1. Spend Spike Detective
- **Tough concept addressed**: Tuning anomaly detection to balance noise vs. missed incidents.
- **Experience flow**:
  1. Users start with a dull cumulative spend graph. They configure anomaly monitors (sensitivity, look-back, threshold) via numeric inputs.
  2. A simulated data stream plays forward, showing when alerts would trigger. Users must choose response actions (contain workload, notify finance) from short-form options.
  3. The module concludes with a dashboard comparing prevented waste vs. false positives, underscoring the ROI of calibrated monitoring.
- **Learning punchline**: Smart anomaly thresholds + rehearsed responses directly convert to avoided GPU waste.

## 2. Invoice Mystery Solver
- **Tough concept addressed**: Reconciling complex AI invoices with FinOps allocations.
- **Experience flow**:
  1. Present users with an anonymized invoice table containing ambiguous line items. They enter mapping rules (tag keys, cost centers) to classify each entry.
  2. A running reconciliation meter tracks how much of the invoice is accounted for; discrepancies trigger hints about missing metadata or contract nuances.
  3. Once reconciled, the interface reveals optimization insights (unused commitments, rate anomalies) as sparkline charts.
- **Learning punchline**: Invoice fluency exposes both compliance gaps and negotiation levers.
