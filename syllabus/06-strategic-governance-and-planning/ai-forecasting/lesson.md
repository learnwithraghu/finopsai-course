# AI Forecasting

## Purpose
Maintain forward-looking views of AI spend that incorporate adoption velocity and contractual commitments.

## Learning Objectives
- Select forecasting models suited for AI (driver-based, rolling averages, machine learning)
- Incorporate pipeline visibility (models in development, staged launches) into forecasts
- Communicate forecast variance and mitigation plans to executives

## Discussion Topics
- Forecast horizons (monthly, quarterly, annual) and data granularity
- Combining committed spend with on-demand scaling expectations
- Variance drivers: demand spikes, vendor price changes, architectural shifts

## Practical Activities
- Build a rolling forecast using historical spend plus pipeline inputs
- Run sensitivity analysis on parameter changes such as GPU rate or adoption curve
- Prepare an executive summary explaining forecast accuracy and risks

## Artifacts & Metrics
- Forecast model workbook
- Sensitivity analysis output
- Executive summary deck
