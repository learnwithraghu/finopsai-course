# Interactive HTML Concepts

## 1. Data Gap Explorer
- **Tough concept addressed**: Diagnosing fragmented AI cost telemetry pipelines.
- **Experience flow**:
  1. Users see a cluttered canvas of disconnected data sources (CUR exports, GPU telemetry, ML metadata) with greyed-out KPIs.
  2. Selecting connectors (Athena, Glue, custom ETL) animates data flows; the page highlights latency, completeness, and lineage scores.
  3. When all quality checks pass, KPIs light up on the right, showing how reliable data unlocks trustable reporting.
- **Learning punchline**: Holistic FinOps reporting depends on deliberate data engineering, not just dashboards.

## 2. Reporting Audience Switcher
- **Tough concept addressed**: Tailoring insights to different stakeholders.
- **Experience flow**:
  1. The page loads with a generic dashboard that feels irrelevant. Users pick an audience (CFO, product GM, ML lead) and key outcomes they care about.
  2. Charts morph in real time: waterfall for CFO, unit cost trend for product, utilization heatmap for engineers. Tooltips explain why each view resonates.
  3. Users can export the narrative as a "talk track" snippet, reinforcing storytelling.
- **Learning punchline**: FinOps reports must adapt language and metrics to the consumer to drive action.
